## Keyboard Shortcuts

*  <kbd>cntrl</kbd> & <kbd>space</kbd> toggles **Content Drawer**. This works while in other windows.
*  <kbd>end</kbd>key snaps object to collide with surface below.

## UI / Layout

* For a **Place Actors** window go to `Edit | Place Actor's Panel`.
